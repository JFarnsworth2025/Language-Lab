# Language Lab

Language Lab is an open-source Python application focused on practical language learning through real-world situations.

Instead of relying on timers, streaks, or point systems, Language Lab encourages repetition until the learner becomes comfortable with new vocabulary.

> **Current Status:** Early Alpha (Active Development)

---

## Current Features

### Lessons

Currently available lesson categories:

- ☕ Coffee Shop
- ✈️ Airport
- 🍽️ Restaurant

Each lesson teaches vocabulary one word at a time through interactive typing exercises.

### Languages

Current vocabulary translations are available for:

- English
- Spanish
- Japanese
- French
- German
- Arabic
- Dutch
- Russian

### Learning Experience

- Interactive word-by-word lessons
- Unlimited attempts until the correct answer is entered
- Case-insensitive answer validation
- Progress tracking within each lesson
- Switch between supported languages without restarting the application

### Architecture

- Modular lesson engine
- Data-driven vocabulary system
- Easy to add new lessons and languages

## Project Philosophy

Language Lab is built around one simple idea:

> **Learning a language shouldn't feel like taking a test.**

Mistakes are part of learning.

Rather than losing points or lives, learners are encouraged to keep practicing until they can confidently spell and recognize each word.

The goal is understanding — not punishment.

---

## Project Goals

Language Lab aims to become a comprehensive language-learning platform built around practical, everyday situations.

Instead of memorizing isolated vocabulary, learners will practice words and phrases they can immediately use in real conversations and daily life.
